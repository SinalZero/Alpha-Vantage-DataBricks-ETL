## Pipeline de Cotações com Databricks

Este projeto implementa um pipeline de dados utilizando Databricks, Spark e Delta Lake,
seguindo a arquitetura Bronze, Silver e Gold.

### Fluxo
- API: ingestão de dados da Alpha Vantage
- Bronze: armazenamento cru com rastreabilidade
- Silver: limpeza e padronização
- Gold: visões analíticas para consumo

### Tecnologias
- Databricks
- Apache Spark
- Delta Lake
- SQL
- Python
